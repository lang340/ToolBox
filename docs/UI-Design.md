```
+--------+----------------------------------------+
| 左侧   |            内容区域                      |
| 导航栏  |                                        |
|        |                                        |

```

## 3. 左侧导航栏

### 3.1 结构

```
+--------+
|        |
|  番茄   |  ← SVG 线条图标
|  钟图标  |
|        |
|  剪贴   |
|  板图标  |
|        |
|  日程   |
|  任务图标 |
|        |
|  日记   |
|  本图标  |
|        |
| ------  |  ← 分隔线
|        |
|  设置   |  ← 设置齿轮图标（P1）
|        |
+--------+
```

### 3.2 图标规范

- 风格：线条风格 (stroke-based)
- 尺寸：24×24px viewBox，渲染尺寸 20×20px
- 线宽：1.5px stroke
- 默认态：`--khaki-500` 描边
- 悬停态：`--khaki-300` 背景 + `--khaki-700` 描边
- 选中态：`--khaki-200` 背景 + `--khaki-800` 描边
- 圆角：图标容器 8px 圆角

### 3.3 交互

- 单击切换工具
- 选中态有浅色背景高亮
- Tooltip：悬停显示工具名称

## 5. 剪贴板记录 UI

```
+----------------------------------------+
|  剪贴板记录            [开关 ●]          |
|                                        |
|  [搜索框]                               |
|                                        |
|  ┌──────────────────────────────┐      |
|  │ 📋 复制的文字内容...           │      |
|  │    10:23:45                   │      |
|  └──────────────────────────────┘      |
|  ┌──────────────────────────────┐      |
|  │ [缩略图]   10:25:12          │      |
|  │            点击查看大图        │      |
|  └──────────────────────────────┘      |
|  ┌──────────────────────────────┐      |
|  │ 📋 另一段文字...              │      |
|  │    10:30:01                   │      |
```

- 开关：toggle 开关组件

- 点击缩略图：弹出 modal 显示大图，ESC 或点击遮罩关闭
- 记录按时间倒序排列

## 8. 通用组件规范

### 8.1 按钮

| 类型 | 样式 |
|------|------|
| 主按钮 | `--khaki-800` 背景，白色文字，8px 圆角 |
| 次按钮 | 透明背景，`--khaki-500` 边框，`--khaki-700` 文字 |
| 文字按钮 | 无边框无背景，`--khaki-600` 文字，hover `--khaki-800` |

### 8.2 输入框

- 背景：`--khaki-100`
- 边框：`--khaki-300` 0.5px
- 聚焦边框：`--khaki-500` 1px
- 圆角：6px
- 高度：32px

### 8.3 卡片

- 背景：`--khaki-100`
- 边框：`--khaki-300` 0.5px
- 圆角：10px
- 内边距：12px

### 8.4 字体

- 标题：15px / 500 weight / `--khaki-800`
- 正文：13px / 400 weight / `--khaki-700`
- 辅助文字：12px / 400 weight / `--khaki-600`
- 字体族：`-apple-system, "Microsoft YaHei", sans-serif`

### 8.5 间距

- 模块间距：16px
- 元素间距：8px
- 内容区内边距：20px
