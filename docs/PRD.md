## 1. 产品概述

### 1.1 定位
Windows 桌面端绿色便携工具箱，双击即用，无需安装。集成番茄钟、剪贴板记录、日程任务、日记本四大工具。

### 1.2 目标用户
需要轻量级桌面效率工具的个人用户。

### 1.3 核心价值
- 便携：单文件夹，解压即用
- 本地优先：所有数据本地存储，无需联网
- 低干扰：线条风格 UI，专注内容

## 3. 非功能需求

| 类别 | 需求 | 标准 |
|------|------|------|
| 性能 | 冷启动时间 | ≤ 3 秒 |
| 性能 | 工具切换响应 | ≤ 200ms |
| 存储 | 数据持久化 | 全部本地，关闭重启不丢失 |
| 存储 | 剪贴板图片存储 | 存入本地文件夹，非数据库 |
| 便携 | 绿色运行 | 单文件夹，无需安装，可放 U 盘运行 |
| 便携 | 数据跟随 | 数据文件与程序在同一目录下 |
| 窗口 | 自由调整 | 可拖拽自由调整窗口大小 |
| 窗口 | 最小尺寸 | 最小宽度 600px，最小高度 400px |

## 4. 用户故事

### US-01 番茄钟
> 作为一个需要专注工作的用户，我想设置一个 25 分钟的番茄钟，时间到了听到提示音，这样我可以保持专注节奏。

### US-02 剪贴板
> 作为一个经常复制粘贴的用户，我想打开剪贴板记录开关后，所有复制的内容（文字和图片）自动保存下来，方便之后查找复用。

### US-03 日程任务
> 作为一个需要规划每天工作的用户，我想在日历上看到我的任务安排，创建循环任务（如每周一开会），打勾标记完成。

### US-04 日记本
> 作为一个喜欢写日记的用户，我想在日记里加粗重点文字、改变颜色、插入图片，按日期查看之前的日记。

## 5. 验收标准

### P0 功能验收
- [ ] 四个工具均可正常打开和使用
- [ ] 番茄钟可自定义时长，倒计时正常，提示音可播放
- [ ] 剪贴板开关可控，文字图片均可记录，图片缩略图可点击放大
- [ ] 任务可增删改查，四种循环规则均生效，任务在日历上正确显示
- [ ] 日记可富文本编辑（加粗/颜色/字号），可插入图片，按日期归档
- [ ] 所有数据关闭后重启仍在
- [ ] 窗口可自由拖拽调整大小

### P1 功能验收
- [ ] 番茄钟工作/休息自动切换
- [ ] 剪贴板单条删除和清空
- [ ] 日记自动保存

---

## 6. 约束与边界

- **不支持**: 云同步、多语言、暗色模式（v1 不做）
- **平台**: 仅 Windows
- **数据**: 全部本地，不联网
