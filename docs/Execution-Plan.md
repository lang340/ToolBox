## 风险预案

| 风险 | 影响 | 预案 |
|------|------|------|
| better-sqlite3 原生模块打包失败 | 便携版无法启动 | 备选 sql.js（纯 JS SQLite，无需编译） |
| 剪贴板图片格式兼容问题 | 部分复制内容无法记录 | 降级为仅记录文字，图片标记为不支持 |
| Quill.js 与 Electron 安全策略冲突 | 富文本编辑器无法初始化 | 备选 ContentEditable 原生方案 |
| electron-builder portable 打包异常 | 无法生成绿色版 | 备选 asar 打包 + 批处理启动脚本 |
